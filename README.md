# Spigot-FeatherJump

An old spitgot plugin that allows you to right click a feather to launch/be lifted upwards.

Compatible with Spigot 1.20.2

This project uses Maven. To build the plugin:

mvn package

The compiled JAR will be inside:

target/FeatherJump-1.0-SNAPSHOT.jar

Build the project or download the compiled JAR.
Place the JAR into your serverâ€™s plugins folder.
Restart the server.
